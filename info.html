This is a site that I built and coded myself after growing tired of the limitations of other site builders and wanting the ability to express myself more.

Firefox is recommended for viewing this site. This site aims to be mobile friendly.

If some parts of the site don't seem to be loading correctly, try CTRL+F5.

Some parts of the site support comments. If you need a comment deleted, please contact me.


DEVELOPMENT

Built with HTML, CSS, and JavaScript. Hosted on Neocities.

Coded in Visual Studio Code. Tested on Firefox for Windows and Safari for iPhone.

Powered by a framework that I developed myself using Python.

Updates are pushed/deployed through the framework using a combination of git hooks and the Neocities CLI.
