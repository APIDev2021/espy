What are 32-bit, 16-bit, 8-bit, 4-bit, 2-bit, and 1-bit images?

What is a bit?

A bit is the smallest unit of data that a computer holds.

Computers are made of millions and millions of bits!

In most computers, bits are represented using binary.

Binary in computers is a piece of data with two different states! In the case with binary, these two different states are typically represented as a 1 or a 0.

A single bit can be written as either a 1 or a 0.


Here's an example of an 8-bit value:

01001011

Each bit is represented as either a 1 or a 0. And there are 8 of these bits.

Thus, this is an 8-bit value!


Bit values are used to represent data! The amount of data they can represent corresponds to the amount of possible states a bit value can have.

A 1-bit value has two possible states, 1 or 0. Thus, a 1-bit value can store 2 pieces of data.

A 2-bit value has four possible states, 00, 01, 10, and 11. Thus, a 2-bit value can store 4 pieces of data.

The amount of data that a bit value can store is as as follows:

2n

With n being the number of bits.

For example, an 8-bit value can store 28 values (which is 256 values) for each pixel.

What do bits have to do with images?

Each pixel in an image is represented by some number of bits.

An 8-bit image has each pixel represented by 8 bits, and a 4-bit image has each pixel represented by 4 bits, and so on.

For example, if we have an image with 4 pixels, and it is an 8-bit image, we can think of those 4 pixels as being represented by 4 8-bit values. Like so:

01001111

01010101

00011101

11011101

DISCLAIMER: Images can often get more complex than this and it is not always as simple as a 8-bit value or so representing a single pixel, this is just a way to comprehend how bit values can represent pixels.

Images often have thousands of pixels, so there are often many many many more bits! This is just an example for only 4 pixels!

As you saw earlier, a single pixel can be represented using 8-bits, but the pixels of an image can also be represented using 4-bits, 32-bits, or even 1-bit!

But what is this data that they are storing?

It's COLORS!

An 8-bit value can store 256 colors, and a 4-bit value can store 16 colors, and so on!

Here's a table to easily reference how many colors each bit value can store.

What can you do with this?

Many images on the internet use 24-bit or higher images. Having 24-bits means there are a lot of possible colors for each pixel, but it can take up a lot of space. Oftentimes, people might convert images into lower bit values to save space. For example, converting a 16-bit image to an 8-bit image.

Of course, with less bits, there are less colors that can be represented! As a result, there are less colors in the image.

This causes some very interesting effects!
